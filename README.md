# MentorMatch

EmpowHER HackDay, Track 1 — EmpowHER General

## Description
MentorMatch promotes gender equality in STEM through meaningful mentorship connections. Our platform uses artificial intelligence (AI) and large language models (LLMs) to match mentors and mentees based on shared goals, skills, and interests — fostering growth, confidence, and inclusion in the tech community.

## Features
Smart Matching Engine – Uses LLMs to read user goals and profiles to suggest ideal mentor/mentee pairings.

AI Icebreaker Generator – Creates personalized conversation starters to help matches connect comfortably.

Inclusive Profiles – Encourages users to highlight their goals, experiences, and preferred mentorship styles.

Dashboard (optional) – View matches, track mentorship goals, and exchange feedback.

Ethical AI Focus – Uses bias-aware matching logic and transparency principles.

## Technologies
HTML for website text and layout
CSS for website styling
JavaScript for integrating frontend and backend
Python for backend and chat implementation
FastAPI for chat response generation
Pydantic for chat request data

Frontend: Next.js (React) + Tailwind CSS
Backend: Node.js / Firebase Functions
Database: Firestore / Supabase
AI: Google Gemini API (or OpenAI GPT-4)
Auth: Google OAuth
Design: Figma → Locofy.ai (React export)
